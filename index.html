Current Research

    Recently the ALS has began the process of upgrading its BPMs. These new BPMs will have the capability to measure turn-by-turn (TBT) data. Currently about 40 have been installed in the storage ring. I am researching the possible applications of the TBT data from these BPMs. To list a few possibilities:

  • Faster tune measurements using mixed TBT BPM data. Current tune measurements only use a single BPM and require many turns. BY using all the BPMs strung together one can get several data points within a single turn and measure a tune value within 2 turn!

  • Calculating phase advances and the beta function using TBT BPM data. Certain techniques allow for the measurement of phase advances between BPMs using only 3 adjacent BPMs. From these phase advances one can get an approximate measurement of the beta function.

  • Using model independent analysis (MIA) techinques to surprise BPM resolution limits. With TBT BPM data one can construct a BPM matrix and break it down using a single value decomposition (SVD). The SVD can then be used to reduce the noise levels in the BPMs and further improve their resolutions.
